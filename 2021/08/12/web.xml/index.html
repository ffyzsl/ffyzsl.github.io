web.xml

web.xml 文件是 Java Web 应用的 XML 格式配置文件， 存放于 WEB-INF 子目录下。web.xml 也称为Java Web 应用的发布描述符文件。

在 web.xml 文件中可以包含如下配置信息：

  • Servlet 的定义
  • Servlet 的初始化参数
  • Servlet 以及 JSP 的映射
  • 安全域配置参数
  • welcome 文件清单
  • 资源引用
  • 环境变量的定义

以上 web.xml 文件的第一行指定了 XML 的版本和字符编码，在 <web-app>根元素中，所有关于 Java Web 应用的具体配置都将加入到这个中。

在 web.xml 中添加 DispatcherServlet 类 加上<servlet> 和 <servlet-mapping>元素

子元素 说明
<servlet-name> 定义 Servlet 的名字
<servlet-class> 指定 Servlet 的完整类名（包括包的名字）
<init-param> 定义 Servlet 的初始化参数
<load-on-startup> 指定当 Servlet 容器启动 Web 应用时，加载各个 Servlet 的次序。

<servlet-mapping>元素用于为 Servlet 映射一个 URL。<servlet-name>子元素指定待映射的 Servlet 的名字，<url-pattern>子元素的值，可以确定访问 DispatcherServlet 的 URL。
