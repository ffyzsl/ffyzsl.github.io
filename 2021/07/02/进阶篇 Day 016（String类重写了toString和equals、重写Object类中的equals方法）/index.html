比较两个字符串，不可以使用“==”，必须调用equals方法

注意：

1、String类已经重写了equals方法，所以比较字符串是否相等时，可以直接使用equals方法

2、String类已经重写了toString方法，所以可以直接输出

总结：

基本数据类型使用”==“判断是否相等，而引用数据类型需要使用equals方法
