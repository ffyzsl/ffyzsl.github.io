PromptTemplate与ChatPromptTemplate详解

发表于 2026-05-04

1. 概述

在 LangChain 中，Prompt 模板是构建 LLM 应用的核心组件。本文介绍四种最常用的模板类：PromptTemplate、ChatPromptTemplate、FewShotPromptTemplate，以及它们之间的区别、联系和常用方法。

2. PromptTemplate（基础字符串模板）

PromptTemplate 是最基础的模板类，用于生成纯文本 prompt。它通过 {variable} 占位符支持变量替换。

常用方法

方法 说明
from_template(template) 类方法，直接从字符串创建模板
format(**kwargs) 传入变量，返回填充后的字符串
invoke(input_dict) LCEL 接口，等同于 format()
partial(**kwargs) 预设部分变量，返回新模板
format_prompt(**kwargs) 返回 PromptValue 对象（字符串 + 消息双模式）

partial() 的使用

partial() 适合固定一部分变量，让调用方只传剩余的变量。

3. ChatPromptTemplate（聊天消息模板）

ChatPromptTemplate 是专门为聊天模型（Chat Model）设计的模板。它由多个消息模板（SystemMessagePromptTemplate、HumanMessagePromptTemplate、AIMessagePromptTemplate）组成，生成的是消息列表而非纯字符串。

常用方法

方法 说明
from_messages(messages) 类方法，从消息列表创建模板
from_template(template) 类方法，从单个字符串创建（自动包装为 human 消息）
format_messages(**kwargs) 返回 List[BaseMessage]
format(**kwargs) 返回字符串（拼接所有消息）
invoke(input_dict) LCEL 接口，返回消息列表
partial(**kwargs) 预设部分变量

消息角色说明

  • ("system", "...") — 系统消息，设定 AI 的行为和角色
  • ("human", "...") — 用户消息，用户的输入
  • ("ai", "...") — AI 消息，对话历史中的 AI 回复
  • MessagePlaceholder — 消息占位符，用于注入历史消息列表

4. FewShotPromptTemplate（少样本提示模板）

FewShotPromptTemplate 在基础模板之上增加了示例（Examples）机制，通过 prefix、examples、suffix 三段式结构组织 prompt。

5. 区别与联系

核心关系图

对比表

维度 PromptTemplate ChatPromptTemplate FewShotPromptTemplate FewShotChatMessagePromptTemplate
生成内容 字符串 消息列表 List[BaseMessage] 字符串（含示例） 消息列表（含示例）
适用模型 LLM（文本补全） Chat Model（聊天） LLM（文本补全） Chat Model（聊天）
消息角色 无 system/human/ai 无 system/human/ai
示例机制 无 无 固定/动态选择 固定/动态选择
format() 返回 str str str 消息列表
format_messages() 不支持 返回消息列表 不支持 返回消息列表
嵌入上层模板 — 支持 MessagePlaceholder 不能 可以直接放入 ChatPromptTemplate

如何选择

  1. 使用文本模型（LLM） → PromptTemplate，需要示例加 FewShotPromptTemplate
  2. 使用聊天模型（Chat Model） → ChatPromptTemplate，需要示例加 FewShotChatMessagePromptTemplate
